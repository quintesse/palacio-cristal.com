Easy Accent

Introduction

Have you ever wondered why you can't enter international characters and special symbols in every Windows program as easily as you can in most word processors like Word for example? Maybe you're thinking now: "but what about using an international key map?". True, that is a possible solution, but one that I was not really satisfied with. Why? Well with an international key map you'll get an international letter every time you type an accent followed by a letter, even if this is not what you wanted. What if you wanted to type 'a good idea'? You'd get á good idea', obviously not what you'd want. The way to prevent this would be by typing a space between the accent and the letter and delete the space afterwards.

This is where Easy Accent comes in: it will let you enter international characters just as easily as you would in Word or Word Perfect but without all the silly problems that you get when using international key maps. You just hold down the CTRL key while typing an accent and a letter and presto!

 

Features

  • Very small and very simple
  • 2 keystrokes are enough
  • Can easily be turned on/off using tray icon
  • Completely configurable keystroke list

Installation

Just extract the ZIP file to whatever directory you want and start the executable, from this moment on Easy Accent will automatically be started whenever Windows is started (unless that is not what you want after which you can just disable it in the options dialog which you can open by double clicking the tray icon).
