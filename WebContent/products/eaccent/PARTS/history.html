History

v1.6
19, March 2000
  • This version is finally able to insert itself into the Windows' startup sequence. An option in the preferences dialog is added for this purpose. 
  • Releasing the control key will now "break" the sequence and clear Easy Accent's internal pattern matching buffer. Before this Easy Accent would still detect sequences even when other keys were pressed in between as long as those keys where typed without holding the control key down. This is now prevented by clearing the buffer each time the control key is released.
